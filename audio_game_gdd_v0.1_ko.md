# 청각 중심 스토리 게임 기획서 v0.1

## 프로젝트 개요
- 가제: **마지막 주파수**
- 플랫폼: Android (Play Store 출시 목표)
- 장르: 오디오 내러티브 퍼즐 어드벤처
- 핵심 차별점: 시각보다 **청각 정보**가 게임 진행의 핵심
- 목표 플레이타임: 40~60분 (챕터 1 기준)

## 한 줄 로그라인
사라진 가족이 남긴 음성 단서를 따라, 플레이어는 소리만으로 진실에 접근한다.

---

## 코어 경험 (Core Experience)
플레이어는 화면의 화려한 비주얼이 아니라,
**방향(좌/우), 거리(가까움/멀어짐), 음색(소리의 성질), 리듬(패턴)**을 읽어 문제를 해결한다.

### 플레이 루프
1. 장면 진입 (짧은 내레이션)
2. 음성/환경음 청취
3. 단서 탐색 (3D 오디오 기반)
4. 퍼즐 해결
5. 선택지 분기
6. 다음 장면 해금

---

## 타깃 유저
- 스토리 중심 인디 게임 유저
- 이어폰/헤드폰 플레이 선호 유저
- 기존 비주얼 퍼즐에 피로를 느낀 유저
- 접근성 높은 경험을 원하는 유저

---

## 챕터 1 시놉시스 (MVP)
주인공은 실종된 동생의 휴대폰 백업에서 이상한 음성 파일들을 발견한다.
파일에는 장소를 특정할 수 있는 배경음(지하철 안내, 신호등, 특정 가게 음악)이 섞여 있고,
플레이어는 이를 추적해 마지막 메시지의 의미를 복원해야 한다.

### 챕터 구조
- Scene 1: 집 (튜토리얼: 좌/우 방향 청취)
- Scene 2: 골목 (거리감 추적)
- Scene 3: 역 주변 (잡음 속 음성 분리)
- Scene 4: 폐건물 입구 (리듬 패턴 잠금해제)
- Scene 5: 옥상 (최종 선택지 + 엔딩 분기)

---

## 퍼즐 디자인 (MVP 5종)

### 퍼즐 1: 방향 추적
- 목표: 특정 음원의 방향 찾기
- 메커닉: 좌/우 팬 값으로 위치 추정
- 실패 조건: 오탐 3회

### 퍼즐 2: 거리 판별
- 목표: 소리의 거리 변화에 맞춰 이동
- 메커닉: 볼륨 + 리버브 양 변화

### 퍼즐 3: 음색 매칭
- 목표: 특정 음색(금속성, 목소리, 전자음)만 골라 선택
- 메커닉: 필터된 사운드 클립 비교

### 퍼즐 4: 리듬 복원
- 목표: 들려준 리듬 패턴을 탭으로 재현
- 메커닉: 4~8비트 패턴

### 퍼즐 5: 잡음 제거
- 목표: 백색소음/도시소음 속 키워드 듣기
- 메커닉: 대역 필터 토글(저역/중역/고역)

---

## 내러티브 분기
- 선택 A: 진실 공개
- 선택 B: 진실 은폐

### 엔딩
- 엔딩 1 (공개): 관계는 무너질 수 있지만 진실은 남는다.
- 엔딩 2 (은폐): 평온은 유지되지만 내적 죄책감이 남는다.

## 접근성/UX 원칙
1. 모든 핵심 소리는 자막 옵션 제공 (온/오프)
2. 헤드폰 권장 안내 필수
3. 소리 단서 반복 재생 버튼
4. 색 약자/저시력 고려한 고대비 UI
5. 진동 피드백(정답 근접 시 약한 햅틱)

## 기술 스택
- 엔진: Unity (LTS)
- 오디오 미들웨어: FMOD
- 분석/크래시: Firebase Analytics + Crashlytics
- 버전 관리: Git + GitHub (private)
- 빌드: Android AAB

### 권장 최소 사양
- Android 9+
- RAM 3GB+
- 이어폰/헤드폰 사용 권장

# DanielOS 개인정보처리방침

최종 업데이트: 2026-03-13

DanielOS(이하 "앱")는 사용자 프라이버시 보호를 중요하게 생각합니다.

## 1. 수집하는 정보
DanielOS는 기본적으로 원격 서버에 개인정보를 수집하지 않습니다.

앱 내부(기기 로컬)에 저장될 수 있는 정보:
- 터미널 출력 로그
- 명령어 히스토리
- 입력창 임시 텍스트(draft)
- 앱 설정값

## 2. 정보 사용 목적
로컬 데이터는 아래 기능 제공을 위해서만 사용됩니다.
- 터미널 세션 표시
- 앱 재실행 시 상태 복구
- 명령 히스토리 탐색
- 로그 저장/복사 기능

## 3. 외부 전송 및 공유
기본 설정에서 사용자 데이터를 제3자와 공유하지 않습니다.

단, 사용자가 SSH/외부 API 등 네트워크 기능을 직접 실행한 경우,
해당 통신은 사용자의 명시적 동작에 의해 발생합니다.

## 4. 보관 및 삭제
- 데이터는 기기 내 앱 저장소에 보관됩니다.
- 앱 삭제 또는 앱 데이터 삭제를 통해 전체 삭제할 수 있습니다.

## 5. 보안
앱은 릴리즈 전 보안 점검, 민감정보 하드코딩 방지 등 기본 보안 원칙을 따릅니다.

## 6. 아동 관련
DanielOS는 아동을 주요 대상으로 설계되지 않았습니다.

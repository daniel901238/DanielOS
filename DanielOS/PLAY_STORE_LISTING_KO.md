# DanielOS Google Play 등록 문구 (국문)

## 앱 이름
DanielOS

## 짧은 설명
안드로이드 개발용 터미널 앱. 로컬 셸, 세션 제어, 로그 관리 지원.

## 전체 설명
DanielOS는 개발 워크플로우를 위한 안드로이드 터미널 앱입니다.

주요 기능:
- 인터랙티브 로컬 셸 세션
- 명령어 히스토리 탐색(이전/다음)
- 세션 재시작/중단
- 로그 저장 및 클립보드 복사
- 앱 재실행 후 로그/입력 상태 복구

개인정보/보안:
- 로컬 우선 저장 방식
- 기본적으로 터미널 데이터를 외부로 자동 업로드하지 않음
- 네트워크 통신은 사용자의 직접 실행 시에만 발생

안내:
- DanielOS는 독립 프로젝트이며, 특정 타 앱의 공식 배포판/후속판을 주장하지 않습니다.
